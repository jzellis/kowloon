Set Up Your Server

This takes about two minutes. You only need to do it once.

    Your Server

    The web address people will use to reach your server. Your domain's DNS should already point at this machine's IP address.

    Shown in the browser title and on the server's landing page.

    Your Admin Account

    Lowercase letters, numbers, underscores.

    Used for admin notifications. Not shown publicly.

    Email (SMTP)

    Optional. Without this, Kowloon won't be able to send notification emails.

    Reverse Proxy & SSL

    By default Kowloon uses Caddy for automatic HTTPS (recommended). Enable this only if you are running your own reverse proxy such as Nginx, Traefik, or Cloudflare Tunnel.

    Kowloon will listen on this port. Your reverse proxy should forward HTTPS traffic here.
